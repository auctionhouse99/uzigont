# Коттеджный посёлок «Узигонт»

Лендинг коттеджного посёлка «Узигонт» — готовые дома с участками 10 соток в Ломоносовском районе Ленинградской области, 3 км от КАД.

## Структура
- `index.html` — разметка (8 секций-экранов + меню)
- `styles.css` — стили, адаптив, анимации
- `script.js` — каталог домов, модалки с галереями, интерактивный генплан на 25 участков, квиз, форма
- `images/` — рендеры, планировки и логотип (оптимизированные JPG/PNG)

## Проекты домов
| Проект | Этажность | Площадь | Спальни / санузлы |
|--------|-----------|---------|-------------------|
| U90 | 1 этаж | 90 м² | 2 / 2 |
| U120 | 1 этаж | 120 м² | 2 / 1 |
| U150 | 2 этажа | 150 м² | 3 / 2 |
| U170 | 2 этажа | 170 м² | 3 / 3 |

## Локальный запуск
Открыть `index.html` в браузере — сборка не требуется (статический сайт без зависимостей).
